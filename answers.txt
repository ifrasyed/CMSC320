
Q1: Why might we consider using Git?

Answer: Git provides many features that make programming easier. It provides memory for code to be saved on. For example, in a case where you computer stops working, you will still have access to 
your repository. Another neat feature that Git has is branching will allows for independent work that is easy to combine with other people's work. 

Q2: What field could data science be applied to? Give an Example.

Answer: Data science can be applied to a variety of fields because of its ability to work
with diverse types of data. It can be applied to healthcare, technology, finance, and muchmore. 
Any field that requires data to be stored in some manner is a field that data science can be applied to. 

Q3: What is the difference between Merging and Rebasing?

Answer: Both merge and rebase solve a similar issue; they both integrate changes from branches but in different ways. The difference between the two is that merging does not obstruct existing branches or change them in any way while rebasing creates new commits for each commit in the original branch.
